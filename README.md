# base

A sandbox website hosted at [base.nikolai.vip](https://base.nikolai.vip).

Built with [Astro](https://astro.build) and deployed automatically via [Netlify](https://www.netlify.com) from the `main` branch.

## Development

```bash
npm install
npm run dev
```

## About

This repo serves as a rotating website sandbox. Each website lives on `main` while active. When a new site is needed, the previous one is archived to its own branch and a fresh project takes over `main`.
